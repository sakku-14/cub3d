## Requirement
- gcc compiler
- MacOS

## Usage
Clone this repository:
```shell
git clone https://github.com/sakku-14/cub3d
```

Generate ***cub3d***:
```shell
make
```

Run ***cub3d***:
```shell
./cub3d maps/map.cub
```
